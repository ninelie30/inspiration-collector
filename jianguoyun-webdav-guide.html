坚果云 WebDAV 同步配置指南

只需 3 步，比 OneDrive 简单得多 — 不需要 Azure 注册，不需要 OAuth，国内直连。

为什么用坚果云？
1. 国内直连，速度最快，不需要代理
2. 注册简单（中国服务，手机号注册即可）
3. WebDAV 协议成熟稳定，不需要 OAuth/App注册
4. 免费版 1GB 上传/月 + 3GB 下载/月，灵感数据几 KB 足够用
5. 数据存在你自己的坚果云里，隐私安全

第一步：注册坚果云（2分钟）

1 打开坚果云官网

访问 www.jianguoyun.com

用手机号或邮箱注册（免费版即可）

如果已有账号，直接登录

已经有坚果云账号？跳到第二步

第二步：获取 WebDAV 应用密码（1分钟）

1 进入安全设置

登录后点击右上角头像 → 设置 → 左侧菜单 安全选项

或直接访问：坚果云安全设置

2 添加第三方应用

找到「第三方应用管理」区域 → 点击「添加应用」

应用名称填：灵感收藏家

点击「生成密码」

3 复制账号和密码

生成后你会看到：

  • 账号：你的坚果云登录邮箱或用户名
  • 密码：一串随机字符（这就是 WebDAV 应用密码）
⚠️ 这个密码不是你的登录密码！是专门给第三方应用用的应用密码，每个应用一个，互不影响。

第三步：在应用中配置（30秒）

1 打开云同步设置

灵感收藏家 → 设置 → ☁️ 云同步 (坚果云) → 打开开关

2 填写账号密码

坚果云账号：填你的邮箱或用户名

WebDAV 应用密码：填第二步生成的密码

3 测试连接

点「测试连接」按钮 → 显示「坚果云连接成功！」即配置完成

数据会自动同步，无需其他操作

4 在其他设备上同步

在其他设备/浏览器打开应用 → 设置 → 启用云同步 → 填入相同的账号和密码 → 测试连接

数据自动双向同步，按时间戳合并，不会丢数据

同步说明

  • 数据存储在坚果云的 /inspiration-collector/data.json 文件中
  • 数据变更后 3 秒自动上传（防抖）
  • 每 60 秒自动拉取一次云端更新
  • 多设备同时编辑时，按时间戳合并，不会丢数据
  • 删除操作会同步到所有设备
  • API Key 等敏感设置不同步，各设备独立配置
  • 离线时正常使用，联网后自动同步

常见问题

测试连接提示「账号或密码错误」

检查：1) 账号是邮箱或用户名（不是手机号）；2) 密码是应用密码不是登录密码；3) 应用密码可能包含特殊字符，复制时注意不要多空格。

手机上能用吗？

可以！坚果云 WebDAV 不需要 HTTPS/OAuth，手机通过局域网访问电脑上的服务器（http://电脑IP:8080）即可同步。如果部署到 Cloudflare Pages 则任何网络都能用。

坚果云免费版够用吗？

完全够用。免费版每月 1GB 上传 + 3GB 下载，灵感数据文件通常只有几 KB，即使每天同步 1000 次也用不到 1MB。

数据安全吗？

数据存在你自己的坚果云中，应用只能访问 /inspiration-collector/ 目录下的文件，无法读取你的其他文件。应用密码可以随时在坚果云设置中删除，删除后同步立即失效。

坚果云会被墙吗？

不会。坚果云是中国公司（上海杳然科技），服务器在国内，WebDAV 接口 dav.jianguoyun.com 国内直连，速度很快。

换了坚果云账号怎么办？

设置中修改账号和密码 → 测试连接即可。但注意：不同账号的坚果云数据不互通，切换后云端是新账号的。

和 OneDrive 方案相比有什么区别？

坚果云：注册简单（手机号）、国内直连、不需要 OAuth/HTTPS、配置只需账号+密码。
OneDrive：需要 Azure 应用注册、需要 OAuth 登录、手机需要 HTTPS 环境。坚果云明显更适合国内用户。

配置完成后，在任何设备打开应用 → 设置 → 启用云同步 → 填账号密码 → 测试连接，数据就自动同步了！
