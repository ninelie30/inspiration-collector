OneDrive 云同步配置指南

只需一次性注册微软应用（获取 Client ID），之后所有设备用微软账号一键登录即可自动同步。

为什么用 OneDrive？
1. 国内可直连（graph.microsoft.com 不需要代理）
2. 纯前端同步，不需要后端服务器
3. 数据存在你自己的 OneDrive 应用文件夹中，隐私安全
4. 微软账号免费，5GB 空间足够用

第一步：注册微软应用

这一步只需要做一次，获取一个 Client ID。

1 打开 Azure 应用注册页面

访问 portal.azure.com → 应用注册

用你的微软账号（@outlook.com / @hotmail.com / @live.com）登录

如果提示没有 Azure 订阅，没关系——注册应用不需要订阅，直接继续

2 点击「新建注册」

在页面顶部点击「+ 新建注册」

3 填写应用信息
  • 名称：随便填，比如「灵感收藏家」
  • 支持的帐户类型：选择 任何组织目录中的帐户和个人 Microsoft 帐户（第三个选项，最通用）
  • 重定向 URI：
    • 平台选 单页应用程序 (SPA)
    • URI 填你的应用访问地址（见下方说明）
重定向 URI 填什么？

就是你在浏览器打开应用时的地址（注意末尾要有 /）：

本地开发：http://localhost:8080/

Cloudflare 部署：https://你的域名.pages.dev/


⚠️ 末尾的 / 必须有！可以添加多个 URI（本地+线上都配上），每行一个

填完后点击「注册」

4 复制 Client ID（应用程序 ID）

注册成功后，在「概述」页面找到 应用程序(客户端) ID

这是一串类似 12345678-abcd-efgh-ijkl-1234567890ab 的字符串

把这个 ID 复制保存好，稍后要填到应用设置中
5 添加 API 权限
  1. 在左侧菜单点击「API 权限」
  2. 点击「+ 添加权限」
  3. 选择「Microsoft Graph」
  4. 选择「委托的权限」(Delegated permissions)
  5. 搜索 AppFolder，勾选 Files.ReadWrite.AppFolder
  6. 搜索 offline_access，勾选 offline_access
  7. 点击「添加权限」
两个权限都要勾：Files.ReadWrite.AppFolder（读写应用文件夹）+ offline_access（保持登录）
6 确认设置（SPA 平台无需额外配置）

如果你选的是 单页应用程序 (SPA) 平台，无需设置「允许公共客户端流」—— SPA 本身就是公共客户端。

如果你的应用注册时只能选 Web 平台（老版界面），则需在「身份验证」→「高级设置」→「允许公共客户端流」选「是」

第二步：在应用中配置

1 填入 Client ID

在灵感收藏家应用中：设置 → ☁️ 云同步 → 启用 → 填入 Client ID → 点「保存」

2 连接 OneDrive

点「连接 OneDrive」按钮 → 跳转到微软登录页 → 登录并授权

授权后会自动跳回应用，显示已连接

3 在其他设备上同步

在其他设备/浏览器打开应用 → 设置 → 启用云同步 → 填入相同的 Client ID → 连接 OneDrive

登录同一个微软账号后，数据自动同步

同步说明

  • 数据存储在 OneDrive 的 应用/灵感收藏家 文件夹下（一个 JSON 文件）
  • 数据变更后 3 秒自动上传（防抖）
  • 每 60 秒自动拉取一次云端更新
  • 多设备同时编辑时，按时间戳合并，不会丢数据
  • 删除操作会同步到所有设备
  • API Key 等敏感设置不同步，各设备独立配置
  • 离线时正常使用，联网后自动同步

常见问题

登录后提示「重定向 URI 不匹配」

检查 Azure 应用中注册的重定向 URI 是否和当前页面地址完全一致（包括 http/https、端口号、路径）。在 Azure 的「身份验证」页面可以添加多个 URI。

手机上能用吗？

可以。但手机必须通过 HTTPS 访问应用（如 Cloudflare Pages 部署的地址），因为微软 OAuth 要求 HTTPS（localhost 除外）。所以手机上用之前需要先部署到 Cloudflare Pages。

OneDrive 在国内会被墙吗？

不会。graph.microsoft.com 和 login.microsoftonline.com 在国内都可以正常访问，不需要代理。OneDrive 个人版的同步功能在国内正常可用。

数据安全吗？

数据存在你自己的 OneDrive 中，应用只能访问自己的专用文件夹（AppFolder），无法读取你的其他文件。Client ID 是公开的，不构成安全风险——PKCE 流程保证了只有你的应用能用这个 ID。

换了微软账号怎么办？

断开同步 → 重新连接 → 登录新账号即可。但注意：不同账号的 OneDrive 数据不互通，切换账号后云端数据是新账号的。

注册应用收费吗？

完全免费。Azure AD 应用注册不需要任何订阅，个人微软账号即可注册。

Cloudflare Pages 部署（手机使用需要）

手机上使用 OneDrive 同步需要 HTTPS 环境，Cloudflare Pages 免费提供：

1 登录 dash.cloudflare.com
2 Workers & Pages → Create application → Pages → Connect to Git
3 选择 GitHub 仓库 ninelie30/inspiration-collector
4 Build command 留空，Build output directory 留空，点 Deploy
5 部署完成后获得 https://xxx.pages.dev 地址
6 把这个地址添加到 Azure 应用的重定向 URI 中
配置完成后，在任何设备打开应用 → 设置 → 启用云同步 → 填 Client ID → 连接 OneDrive，数据就自动同步了！
